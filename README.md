# Neon Overlay

Official website and downloads for Neon Overlay.

## Live Website

After enabling GitHub Pages, your site will be available at:

`https://trxpworks.github.io/neon-overlay/`

## Publish Steps

1. In this repository, open **Settings -> Pages**.
2. Under **Build and deployment**, choose **Deploy from a branch**.
3. Select branch **main** and folder **/(root)**.
4. Save.

## Release Assets Required

The website download buttons point to the latest release assets with these exact names:

- `Neon-Overlay-Portable.exe`
- `Neon-Overlay-Setup.exe`

To publish a new version:

1. Go to **Releases -> Draft a new release**.
2. Upload those two EXE files using the exact names above.
3. Publish release.

The website will automatically download the newest release from:

- `https://github.com/trxpworks/neon-overlay/releases/latest/download/Neon-Overlay-Portable.exe`
- `https://github.com/trxpworks/neon-overlay/releases/latest/download/Neon-Overlay-Setup.exe`
